# BFVInfo WeChat App
微信战地5战绩查询小程序，目前测试阶段，还未上线。

前端基于微信API，没怎么做（主要是`css`不太会）。

后端用的`flask`，提供了一个查询接口，仅此而已。

数据是从[BF Tracker](https://battlefieldtracker.com/)爬的，提取了一部分。
